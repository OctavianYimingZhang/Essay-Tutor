# Drafting And Critical Analysis

Use this reference after plan approval or when revising essay prose.

## Draft From The Approved Plan

The approved plan controls:

- thesis;
- scope;
- section hierarchy;
- paragraph functions;
- citation strategy;
- figure/table/data strategy;
- critical-thinking targets;
- word-count allocation.

If research after approval contradicts the plan, pause drafting and present a revision note rather than silently changing the essay.

## Paragraph Logic

Default paragraph:

```text
Claim -> Mechanism/Evidence -> Interpretation -> Scope or Limitation -> Link back
```

Evidence-heavy paragraph:

```text
Evidence -> Mechanism tested -> Result -> Interpretation -> Limitation
```

Comparison paragraph:

```text
Shared problem -> Comparison axis -> Model A -> Model B -> Evidence -> Evaluation
```

Application paragraph:

```text
Problem -> Mechanism or theory -> Intervention or case -> Evidence -> Boundary -> Implication
```

Each paragraph should have one main function. Do not use a paragraph as a storage container for loosely related facts.

## Descriptive And Analytic Balance

Minimum target:

```yaml
AnalyticMinimum:
  introduction: 20_percent
  main_body: at_least_30_percent
  discussion: at_least_70_percent
  conclusion: 50_percent
```

Preferred whole-essay balance:

```yaml
PreferredEssayBalance:
  descriptive: 40_to_60_percent
  analytic: 40_to_60_percent
```

Descriptive content:

- defines terms;
- states mechanisms;
- reports evidence;
- describes methods;
- gives source-backed examples.

Analytic content:

- explains why a mechanism matters;
- states what evidence proves or fails to prove;
- compares models;
- defines scope or limitations;
- calibrates causal strength;
- links evidence to the essay question;
- identifies clinical, theoretical, or methodological implications.

A paragraph fails if it has more than two consecutive descriptive sentences without analysis.

## Section Rules

Introduction:

- start with the problem or thesis, not metacommentary;
- define only necessary terms;
- set the boundary of the essay;
- state the argument directly;
- avoid "This essay will..." when a direct thesis is possible.

Main body:

- use subtitles when they improve navigation;
- make each subtitle an argument unit, not a topic label only;
- integrate evidence and interpretation adjacent to each other;
- avoid lists of mechanisms, molecules, papers, or treatments without analytic use.

Discussion:

- synthesise the main body rather than repeating it;
- evaluate evidence quality, model fit, methodological limits, translation, uncertainty, and alternative explanations;
- identify future directions only when they follow from stated limits.

Conclusion:

- answer the question directly;
- synthesize the thesis and implications;
- add no new evidence, source, mechanism, or claim.

## Language Rules

Use academic prose that is direct, precise, and evidence-calibrated.

Required:

- Start with the answer or problem.
- Prefer direct positive claims.
- Keep model, mechanism, evidence, and implication distinct.
- Use examples as evidence for a wider claim.
- Convert experiments into evidence, interpretation, and limitation.
- Compress repeated definitions, repeated claims, decorative transitions, and low-value background.
- Preserve causal strength, scope qualifiers, and model boundaries when shortening.
- Cite non-obvious factual claims, theories, mechanisms, experiments, data, or generalisations.

Avoid:

- lecture-route or source-route narration;
- process commentary inside the essay body;
- decorative transitions;
- broad claims such as "this is important" without a specified consequence;
- unsupported statistics;
- citation stacking;
- negative-only openings that are immediately corrected in the next sentence;
- true but unneeded detail that makes the answer encyclopedic rather than argumentative.

## Critical Thinking Checklist

Before finalizing, verify that the essay:

```yaml
CriticalThinkingCheck:
  thesis_is_arguable: true
  evidence_is_interpreted: true
  models_are_compared_where_relevant: true
  limitations_affect_the_argument: true
  causality_is_calibrated: true
  alternative_explanations_considered: true
  implications_follow_from_evidence: true
  conclusion_synthesises_not_repeats: true
```

If critical thinking is weak, do not add a detached "critical sentence" at the end. Rewrite the paragraph so evidence and interpretation are adjacent.
