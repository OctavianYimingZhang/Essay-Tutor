# CourseWork Killer Critical Analysis

Use this focused skill to add evaluative reasoning without weakening evidence accuracy.

## Shared Resource Rule

Read shared resources from the first existing CourseWork Killer root:

- packaged path: `../../`
- local copied-skill path: `../coursework-killer/`
- default local path: `~/.codex/skills/coursework-killer/`

## Workflow

1. Read `references/drafting-and-critical-analysis.md`.
2. Read `references/critical-writing-bank.md` when the task needs concrete evaluative moves or sentence patterns.
3. Read `references/research-and-citation.md` when critique depends on source quality, method limits, evidence hierarchy, or citation support.
4. Build or revise a CriticalAnalysisPlan with explicit body-paragraph or Discussion insertion points.
5. Tie each critical move to a claim, method, source boundary, theoretical limit, comparison, implication, or alternative explanation.
6. Keep critique proportional to the assignment purpose and evidence base.
7. Read `references/qa-and-validation.md` before final delivery.

Do not add generic scepticism or unsupported limitations.
