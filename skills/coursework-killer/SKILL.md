# CourseWork Killer Plugin Router

This is the plugin-discoverable router entrypoint. It preserves the existing workflow contract by delegating to the package root [`../../SKILL.md`](../../SKILL.md).

Before doing coursework work, read and follow [`../../SKILL.md`](../../SKILL.md), then route to the focused Skill that matches the confirmed coursework task.
