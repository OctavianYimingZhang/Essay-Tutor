# Essay Tutor Research Citation

Use this focused skill to build a source-backed evidence map and citation strategy.

## Shared Resource Rule

Read shared resources from the first existing Essay Tutor root:

- packaged path: `../../`
- local copied-skill path: `../essay-tutor/`
- default local path: `~/.codex/skills/essay-tutor/`

## Workflow

1. If the brief is not locked, route through `essay-tutor-intake-planning` first.
2. Read `references/research-and-citation.md`.
3. Read `references/intake-and-planning.md` only when citation quantity, source base, or format requirements are unresolved.
4. Prioritise the locked brief, rubric, official course material, required readings, user files, authoritative academic sources, and verified external literature.
5. Build a claim-to-source map before drafting claim-heavy sections.
6. Use claim-led parenthetical citation placement by default unless the brief or user requires another style.
7. Verify source metadata before final references; mark unsupported claims or missing metadata as evidence gaps.
8. Read `references/qa-and-validation.md` before final citation or reference-list delivery.

Do not invent statistics, mechanisms, source metadata, or reference details.
