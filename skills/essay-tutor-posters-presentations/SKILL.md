# Essay Tutor Posters Presentations

Use this focused skill for visual academic communication in posters and presentations.

## Shared Resource Rule

Read shared resources from the first existing Essay Tutor root:

- packaged path: `../../`
- local copied-skill path: `../essay-tutor/`
- default local path: `~/.codex/skills/essay-tutor/`

## Workflow

1. Read `references/intake-and-planning.md` and `references/visuals-tables-data.md`.
2. Use `python3 scripts/build_intake_questions.py poster-plan` when poster canvas, message hierarchy, or visual/data assets are unresolved.
3. Use `python3 scripts/build_intake_questions.py presentation-plan` when timing, audience, assessment mode, notes, or script expectation is unresolved.
4. Build a poster story plan or slide storyboard instead of an essay outline.
5. Map every panel or slide to a purpose, evidence basis, visual asset, citation placement, and reader or audience takeaway.
6. Read `references/drafting-and-critical-analysis.md` for poster panel text, slide text, speaker notes, and interpretation.
7. Read `references/qa-and-validation.md` before final delivery.

Keep visuals, citations, and spoken or display text aligned with the locked brief.
